# 🚀 Dynamo Skips App

Dashboard for analyzing PyTorch Dynamo CPython test results and graph break reasons.

## Overview

This Streamlit application provides interactive visualization of:
- CPython test results across 38 modules (4,352+ tests)
- Graph break reasons and failure analysis
- Pass/skip rates by module
- Detailed test breakdowns

## Running CPython Tests

Generate fresh test data using the test runner script:

### Quick Start (All Tests)

Or directly:
```bash
python cpython_test_runner.py
```

## Data Sources

The dashboard reads from:
- `data/all_tests_output_*.txt` - Full test output logs

Data is generated by running tests (see "Running CPython Tests" section above).

**Filename Format**: `all_tests_output_<11-char-commit>_<YYYYMMDD>.txt`
- Commit hash included for version tracking
- Date only (not timestamp) for cleaner filenames


### Self-hosted

```bash
pixi run streamlit run app.py --server.port 8501 --server.address 0.0.0.0
```

## Keeping Dependencies in Sync

Use Pixi as the source of truth and regenerate `requirements.txt` before deploy:

```bash
pixi run sync-requirements
```

This exports dependencies from `pixi.toml` (`[dependencies]` and `[pypi-dependencies]`) into `requirements.txt`.
